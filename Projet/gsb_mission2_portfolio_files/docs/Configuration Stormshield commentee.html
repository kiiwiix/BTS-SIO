Configuration Stormshield commentée

Interfaces VLAN

Chaque VLAN interne dispose d’une interface sur le Stormshield avec une passerelle en .254, par exemple 192.168.250.254/24 pour le VLAN serveurs.

Routage

La route par défaut envoie les destinations inconnues vers la passerelle WAN 120.121.122.1 via l’interface out.

NAT

Le NAT permet aux adresses privées des VLAN internes de sortir vers Internet avec l’adresse WAN du Stormshield.

Filtrage

Les règles contrôlent les communications entre VLAN et vers Internet. Le VLAN visiteurs est limité aux services essentiels comme DNS, DHCP et accès Internet.
