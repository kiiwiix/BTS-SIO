Fiche de procédure - Mission 3 AD / DNS / DHCP

  1. Préparer les serveurs dans le VLAN 250.
  2. Installer le rôle AD DS sur LABANNU et créer le domaine swiss-galaxyB2.com.
  3. Ajouter LABANNUBIS au domaine et le promouvoir en contrôleur de domaine secondaire.
  4. Configurer la zone DNS interne et le redirecteur DNS vers 8.8.8.8.
  5. Installer et configurer le service DHCP sous Debian avec les étendues par VLAN.
  6. Tester depuis un client Windows : ipconfig, nslookup, ping, dcdiag et réplication.
